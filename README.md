# Custom Probe-Based Spatial Transcriptomics Enables Microbiome Detection in FFPE Colorectal Cancer Tissue

This repository contains the probe design pipeline supporting Aburajab et al., City of Hope.

## Overview

The pipeline is designed to create custom 16S rRNA split-ligation probe panels targeting microbiome species for use with the 10x Genomics Visium CytAssist platform.
It takes FASTA sequences for each target species, generates consensus sequences via multiple sequence alignment, and designs probe pairs that pass QC filters derived from the 10x Genomics probe design specification.

## Dependencies

## Usage

​```bash
WD_SEQ="/path/to/visium_microbiome" source run_probe_pipeline.sh
​```

### Step 1: Consensus sequence generation

For each species in `accessions.tsv`, combines all accession FASTAs, aligns them with MAFFT, and generates a consensus sequence by taking the most common non-gap base at each position.

### Step 2: Probe design and aggregation

Designs split-ligation probe pairs from consensus FASTAs. Each probe pair consists of a left half and right half, each 25 bp, tiling the reverse complement of the consensus sequence. Probes passing all QC filters are written to per-species TSVs, then aggregated into a single Excel file (one sheet per genus) and a FASTA file per genus.
